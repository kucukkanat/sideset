# Notifying the user

## How to use ntfy.sh

Send notifications via:
```bash
curl -d "message" https://ntfy.sh/YOUR_TOPIC
```

Or with title:
```bash
curl -H "Title: Your Title" -d "message" https://ntfy.sh/YOUR_TOPIC
```

## Notifying the user
Notify the user at ntfy.sh/tolga-7509-agent when user asks you to notify them
